NodePK: Neural ODE PK App Documentation

Overview

This app predicts drug concentration-time profiles for patients using a Neural ODE-based pharmacokinetic model. It also calculates PK metrics and visualizes variability across patients.

Input CSV File Format

The CSV file must contain the following columns:

Column Description Example Unit
PatientID Unique identifier for each patient/subject. P001 N/A
Time Time point when the drug concentration was observed. 0, 0.5, 1, 2... hours (h)
Dose Amount of drug administered to the patient at time 0. In this study, all patients can have the same dose to assess variability in response. 100 mg (milligrams)
Observed Measured drug concentration in the patient's plasma at the corresponding time point. Used to assess variability and compare against model predictions. 88.2 ng/mL (nanograms per milliliter)

Output Metrics

The app calculates the following pharmacokinetic metrics:

Metric Description Unit
Cmax Maximum predicted plasma concentration after dose. ng/mL
Tmax Time at which Cmax occurs. hours (h)
AUC Area under the concentration-time curve, representing overall drug exposure. ngÂ·h/mL
HalfLife Elimination half-life: time required for plasma concentration to reduce by half. hours (h)
Clearance Apparent drug clearance from plasma (volume of plasma cleared per unit time). L/h

Graph

The app generates a graph of log(concentration) vs. time for each patient, allowing you to visualize pharmacokinetic profiles and inter-individual variability.
